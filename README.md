En el commit de classe, vaig poder comprobar com em funcionava tot el Rest-Api i en principi no donava errors ninguna part del codi.

El projecte utilitza un sistema LIFO, per tant quan els drons esitguin dins del magatzem, el primer en sortir (reparat) sera l'ultim que hagi entrat.

Es permet crear/actualitzar un dron (indicant la id, fabricant, hores de vol, nom i model), crear/actualitzar un pilot (id, nom, cognom, hores de vol) i una crear una reserva (amb la info del dron, pilot
posició i durades)

També permet consultar la situació dels drons (poguent fer que aquests estiguin reparats o per reparar), pilots i reserves.
